# 💰 Expense Tracker (Python GUI App)

A modern and user-friendly **Expense Tracker Application** built using **CustomTkinter**.
This app helps users manage their daily expenses with a clean UI and powerful features like data storage and graphical analysis.

---

## 🚀 Features

* ➕ Add Expenses (Name, Amount, Category, Date)
* 📋 View Expenses in Scrollable List
* ❌ Delete Individual Expenses
* 💵 Real-time Total Balance Calculation
* 💾 Data Persistence using JSON (auto save & load)
* 📊 Visualize Expenses with Graphs:

  * Pie Chart (Distribution)
* 🎛️ Toggle between Bar & Pie Graphs

---

## 🖼️ UI Preview

* Clean Dark Mode Interface 🌙
* Card-based Expense Display
* Scrollable Expense List
* Dashboard-style Total Balance + Graph Controls

---

## 🛠️ Tech Stack

* Python 🐍
* CustomTkinter (GUI)
* Matplotlib (Data Visualization)
* JSON (Data Storage)

## 📊 How It Works

* User adds expense details via input fields
* Data is stored in a JSON file (`Cards.json`)
* Expenses are displayed dynamically in the UI
* Total balance updates automatically
* Graphs are generated based on category-wise data
