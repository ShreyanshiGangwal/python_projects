# 🚀 Smart To-Do App (Python GUI)

A modern, minimal, and feature-rich **To-Do List Application** built using **CustomTkinter**.
Designed with a clean UI and powerful task management features to boost productivity.

---

## ✨ Features

* ✅ Add Tasks with Title, Description, Date & Priority
* ✏️ Edit Existing Tasks
* ❌ Delete Tasks
* ✔️ Mark Tasks as Completed
* 🔍 Search Tasks by Title
* 📊 Progress Tracker (Completed vs Total Tasks)
* 💾 Auto Save & Load Tasks using JSON
* 🎨 Dark Mode UI with Modern Design

---

## 🖥️ Tech Stack

* **Python**
* **CustomTkinter** (Modern GUI)
* **Tkinter MessageBox**
* **JSON** (Data Storage)
* **OS Module**

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/ShreyanshiGangwal/python_projects.git
cd python_projects
```

### 2. Install Dependencies

```bash
pip install customtkinter
```

### 3. Run the App

```bash
python Todo_list.py
```

---

## 🧠 How It Works

* Tasks are stored in a **JSON file** (`tasks.json`)
* Each task contains:

  * Title
  * Description
  * Date
  * Priority
  * Completion Status
* UI dynamically updates using **CustomTkinter widgets**
