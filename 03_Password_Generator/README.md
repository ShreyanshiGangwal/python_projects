# 🔐 Password Generator (CustomTkinter GUI)

A modern and stylish **Password Generator Application** built using **Python & CustomTkinter**.
This app allows users to generate secure, customizable passwords with a beautiful dark-themed UI.

---

## ✨ Features

* 🔐 Generate strong & secure passwords
* 🎚️ Adjustable password length (slider)
* ☑️ Custom character selection:

  * Uppercase letters (A-Z)
  * Lowercase letters (a-z)
  * Numbers (0-9)
  * Symbols (!@#$%)
* 📋 One-click copy to clipboard
* 📊 Password strength indicator (Progress Bar with colors)
* 🎨 Modern dark UI with neon glow styling

## 🛠️ Tech Stack

* **Python**
* **CustomTkinter** (for GUI)
* **pyperclip** (for clipboard functionality)
* Built-in libraries:

  * `random`
  * `string`
  
---

## 💡 How It Works

1. Select desired password length using slider
2. Choose character types (uppercase, lowercase, numbers, symbols)
3. Click **Generate Password**
4. View password strength via color-coded progress bar
5. Copy password instantly using the Copy button
