# 🔄 Unit Converter App

A modern and user-friendly **Unit Converter Desktop Application** built using **Python** and **CustomTkinter**.
This app allows users to easily convert values between different units with a clean and attractive UI.

---

## ✨ Features

* 🔢 Convert between multiple unit categories:

  * Length (m, km, cm)
  * Weight (kg, g)
  * Temperature (°C, °F)

* 🎨 Modern Dark UI with custom color theme

* 🔁 Swap button to quickly switch units

* ⚡ Fast and accurate conversions

* 🛡️ Error handling for invalid inputs

* 📱 Clean and responsive layout
